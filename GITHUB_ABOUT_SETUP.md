# GitHub Repository Setup Guide

## 📝 Set Up "About" Section

### Step 1: Go to Your Repository

Open: https://github.com/shamanthwick/tamarin-rule-generator

---

### Step 2: Add Description & Topics

**On the right side of your repository page**, you'll see the **"About"** section.

**Click the ⚙️ (gear icon)** next to "About"

---

### Step 3: Add Description

In the **"Description"** field, paste:

```
AI-powered tool that converts protocol diagrams (PNG, PDF) into formal Tamarin verification models. Task 1 submission for IIT Roorkee Formal Verification course.
```

Click **"Save"**

---

### Step 4: Add Topics

In the **"Topics"** field, add these (one at a time):

```
tamarin-prover
formal-verification
security-protocols
ai-tools
ocr
python
iit-roorkee
cybersecurity
artificial-intelligence
protocol-verification
```

**How to add:**
1. Type the topic name
2. Press **Enter** or click to select
3. Repeat for each topic
4. Click **"Save"** when done

---

### Step 5: (Optional) Add Website

If you want to add a website link, you can add:
```
https://github.com/shamanthwick/tamarin-rule-generator
```

---

## ✅ Final "About" Section Should Look Like:

```
📖 AI-powered tool that converts protocol diagrams (PNG, PDF) into formal Tamarin verification models. Task 1 submission for IIT Roorkee Formal Verification course.

🏷️ Topics: tamarin-prover, formal-verification, security-protocols, ai-tools, ocr, python, iit-roorkee, cybersecurity, artificial-intelligence, protocol-verification
```

---
