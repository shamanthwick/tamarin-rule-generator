# How to Convert HTML to PDF

## Method 1: Using Browser (Easiest - Recommended)

1. **Open the HTML file in your browser:**
   - Double-click `TASK1_SUBMISSION.html`
   - OR right-click → Open with → Chrome/Edge/Firefox

2. **Print to PDF:**
   - Press `Ctrl + P` (Windows) or `Cmd + P` (Mac)
   - In the printer dropdown, select **"Save as PDF"** or **"Microsoft Print to PDF"**
   - Click **"Save"**
   - Choose location and filename: `TASK1_SUBMISSION.pdf`

3. **Done!** ✅ You now have a beautiful PDF

---

## Method 2: Using Python (If wkhtmltopdf is installed)

```bash
# Install wkhtmltopdf from:
# https://wkhtmltopdf.org/downloads.html

# Then run:
python -c "import pdfkit; pdfkit.from_file('TASK1_SUBMISSION.html', 'TASK1_SUBMISSION.pdf')"
```
